# src-tauri/

Tauri backend (Rust). To be implemented in the future.

## Implementation Notes
- Initialize with `bunx tauri init`
- Add plugins: `bunx tauri add fs` / `bunx tauri add path`
- Transparent background: set `transparent: true`, `decorations: false` in `tauri.conf.json`
- Works with `src/adapters/tauri.ts`
