# @vrm-desktop/core

Platform-independent VRM logic.

## Dependencies
- three.js
- @pixiv/three-vrm
- @pixiv/three-vrm-animation

## Exports
- `initScene()` - Initialize Three.js scene
- `loadVRM()` - Load VRM model
- `AnimationController` - VRMA animation control

## Note
This package does NOT include platform-specific I/O (fetch, Tauri API, etc.)
